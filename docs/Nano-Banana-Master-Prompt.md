# HOF — Master Prompt for Nano Banana PRO (Mockup Image)

Use the prompts below in **Nano Banana PRO** (or similar AI image generators) to generate a **mockup image** of the HOF website. Nano Banana PRO supports accurate text in images and high-quality output—ideal for a single hero or full-page website mockup.

---

## Prompt (full-page website mockup)

```
Professional mockup image of a dark, premium website landing page on a desktop screen. The website is for HOF (House of Fiesta), a nightclub in Cancún, México.

Visible on screen:
- Top: large bold headline text “HOF” in white or light gray, condensed display font.
- Below it: smaller line “House of Fiesta” in gold/amber.
- Tagline in clear, readable text: “Rush the night. Break the rules. Be HOF.” in light gray or white.
- Dark background: pure black or near-black (#0A0A0A). No clutter.
- One prominent button: solid red (#E30613), white text “Get in” or “Reserve”.
- Optional: a second section below with short punchy text “You’re one of us now.” and “Sweat. Glow. Repeat.” on dark gray cards (#1A1A1A).
- Small gold “VIP” or red “Hot” badge somewhere.
- Footer line: “HOF · House of Fiesta · Cancún”.

Style: high-end nightlife, bold, tribal mood. Minimal layout, strong contrast. Red and gold accents on black. Clean sans-serif typography. No distortion; all text sharp and legible. Studio-quality web design mockup, 16:9 or full browser frame.
```

## Tips for Nano Banana PRO

- **Text:** Nano Banana PRO renders text well; spell out exactly what should appear (e.g. “Rush the night. Break the rules. Be HOF.”).
- **Colors:** Name colors (red, gold, black) and use hex (#E30613, #FFA800, #0A0A0A) if the tool accepts them.
- **Aspect ratio:** Use 16:9 or similar for a browser/desktop mockup; or portrait for a mobile mockup.
- **Style:** Phrases like “professional mockup”, “studio-quality”, “clean minimal” help keep the output on-brand.
